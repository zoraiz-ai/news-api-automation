# 📰 News API Automation

An AI-powered News API Automation built using **n8n**, **HTTP Request**, **News API**, and **Gmail Automation**.

---

## 📖 Project Overview

This workflow automatically fetches the latest news articles from a News API, processes the response, and sends formatted news updates through Gmail.

The project demonstrates how APIs can automate real-time news delivery using intelligent workflows.

---

## ✨ Features

- Latest News Headlines
- News API Integration
- HTTP Request Integration
- Gmail Notifications
- Real-Time News Updates
- Easy to Customize

---

## 🛠 Technologies Used

- n8n
- HTTP Request
- News API
- Gmail
- API Automation
